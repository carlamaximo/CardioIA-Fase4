# CardioIA-Fase4
Projeto acadêmico da FIAP desenvolvido para a Fase 4 do CardioIA. A solução utiliza técnicas de Visão Computacional, Redes Neurais Convolucionais (CNN) e Transfer Learning com ResNet50 para classificação de imagens de ECG, auxiliando a interpretação de padrões cardíacos de forma automatizada.
